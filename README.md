# Database
## 專案說明
- 資料量超過Excel可處理的最大列數1,048,576列，因此需要建立資料庫來存取。
- 需要結合2種類型的資料，因此需要透過SQL或Python來合併資料庫。
- 進行探索式資料分析。
- 以視覺化的方式呈現資料分析結果。
